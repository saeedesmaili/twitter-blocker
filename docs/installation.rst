============
Installation
============


Stable release
--------------

To install Twitter Blocker, run this command in your terminal:

.. code-block:: console

    $ pip install twitter_blocker

This is the preferred method to install Twitter Blocker, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Twitter Blocker can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/saeedesmaili/twitter_blocker

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/saeedesmaili/twitter_blocker/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/saeedesmaili/twitter_blocker
.. _tarball: https://github.com/saeedesmaili/twitter_blocker/tarball/master
